# College-chatbot
🎓 AURCC Smart College Chatbot
📌 Overview

The AURCC Smart College Chatbot is an AI-powered web application built using Streamlit and Google Gemini API.
It helps students quickly access important college information like HOD details, subjects, timetable, and procedures.

The chatbot first checks a custom college dataset and then uses AI to answer unknown queries.

🚀 Features
📚 Provides college information instantly
👨‍🏫 Displays Principal & HOD details
🗓️ Shows subjects and timetable
📝 Explains OD & leave procedures
🤖 AI fallback using Google Gemini
💬 Interactive chat UI using Streamlit
🛠️ Tech Stack
Python
Streamlit
Google Generative AI (Gemini API)
dotenv (Environment Variables)
